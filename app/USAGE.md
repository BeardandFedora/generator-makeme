Description:
  Create and manage a `badass` Full-Stack AngularJS + Node app

#### Quick help guide:

##### **Gulp:**
```
gulp serve
gulp test
gulp build
```

##### **Generators:**

  Server Side:
```
yo makeme:endpoint
```

  Client Side:
```
yo makeme:route			# the module
yo makeme:controller
yo makeme:filter
yo makeme:directive
yo makeme:service
yo makeme:provider
yo makeme:factory
yo makeme:decorator
```

  Deployment:
```
yo makeme:openshift
yo makeme:heroku
```
